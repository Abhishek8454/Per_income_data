# Per_income_data
This repository contains per capita income data used to study the average income of individuals over different years. The dataset helps in understanding income growth trends, economic development, and changes in living standards over time. It can be used for data analysis, visualization, and basic machine learning projects. By analyzing this data, users can observe patterns, compare yearly income changes, and gain insights into economic progress. This dataset is suitable for beginners as well as students learning data science concepts such as data preprocessing, plotting graphs, and trend analysis. It serves as a simple and practical example for economic data exploration.
